AR都市構想プロジェクト

AR技術を活用して、都市や地域の課題を解決し、新たな価値を創出することを目指すプロジェクトです。

地域活性化

観光案内・情報発信、文化・歴史のAR化、イベント開発

都市課題解決

移動支援、防災支援、インフラ管理の効率化

新価値創造

エンターテイメント、教育支援、新規ビジネス創出

プラットフォームの特徴

  • マルチデバイス対応（スマートフォン、タブレット、ARデバイス）

基本機能

  • ARKit/ARCore基盤

寄付の種類

企業様からのご支援も受け付けております。金額は応相談となります。

寄付金の報告について

  • 寄付金の使途は、四半期ごとにウェブサイトで公開いたします
  • 年次報告書でも詳細な活動報告と収支報告を行います
  • 定期寄付者には、毎月のニュースレターで活動の進捗をお伝えします

税制上の優遇措置

当団体は認定NPO法人として、以下の税制優遇措置が適用されます：

  • 個人の寄付：所得税の税額控除または所得控除
  • 法人の寄付：一般損金算入枠の拡大
  • 相続財産からの寄付：非課税措置

確定申告に必要な領収書は、ご寄付の後に発行いたします。

開発環境のセットアップ

1. SDK のインストール

npm install @ocn/sdk
# または
yarn add @ocn/sdk

2. 認証設定

const ocn = new OCN({
  apiKey: 'your-api-key',
  projectId: 'your-project-id'
});

3. 基本的な使用例

// ARコンテンツの作成
const arContent = await ocn.ar.createContent({
  type: 'model',
  source: 'path/to/model.glb',
  position: { x: 0, y: 0, z: 0 }
});
