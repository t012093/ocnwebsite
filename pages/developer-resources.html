サンプルコード

SDK初期化

// OCN SDKの初期化
const ocn = new OCN({
    apiKey: 'your-api-key',
    projectId: 'your-project-id'
});

ARコンテンツの作成

// ARコンテンツの作成
const arContent = await ocn.ar.createContent({
    type: 'model',
    source: 'path/to/model.glb',
    position: { x: 0, y: 0, z: 0 }
});

データの取得と分析

// 分析データの取得
const analytics = await ocn.analytics.getData({
    startDate: '2025-01-01',
    endDate: '2025-12-31',
    metrics: ['views', 'interactions']
});
